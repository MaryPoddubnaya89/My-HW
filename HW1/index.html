Определения тегов

Блочные теги

div
Используется в качестве выделения части документа в отдельный бдок. Не несет семантического смысла
ol
Определяет упорядоченный список
ul
Определяет маркерованный список
h1, h2, h3, h4, h5, h6
Заголовки документа
form
определяет блок с формами

Семантические теги

Header
определяет шапку документа
Main
определяет основное содержимое документа
Footer
определяет подвал документа
Article
определяет блок с новостями, блогами, статьями
Section
определяет раздел документа

Одиночные теги

img
определяет изображение
meta
определяет метатеги для хранения информации для поисковых систем
br
тег переноса на другую строку
input
тег для создание текстовых полей в форме на веб-странице
link
тег для подключения внешних файлов стилей и шрифтов
